Aztec Technologies Maps

Web Pages:

The Top Trending Twitter Topic:
Displays the relative volume of Tweets around the UK on the current top trending Twitter topic. Uses Spring's Twitter Template for the searches. The tweet volumes are updated every 5 minutes.

Premiership Football Tweets:
Displays the number of tweets for each Premiership football club. Connects to Twitter via the Streaming API, processing the tweets related to each club. The tweet counts are updated every minute.

GitHub location:

https://github.com/rgolder1/aztec-map

Technologies used:

  • Spring Boot (for rapid development)
  • d3 Javascript library (UI)
  • Gradle (build)
  • Spring Social (Twitter integration):
    • TwitterTemplate searches
    • Twitter's Streaming API
  • Spring RESTful Web Services
  • Deployed on Cloud Foundry
  • Scribe (OAuth authentication)
